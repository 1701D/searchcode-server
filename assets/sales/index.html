How searchcode server works

By indexing your source code it allows you to search over this code quickly, filtering down by repositories, languages and file owners to find what you were looking for.

  • Index any GIT, SVN or file repository public or private
  • Identifies 100+ languages and ignores minified code
  • Filter by repository, language or user
  • Estimates the cost of every file with the COCOMO calculation adjustable to your needs
  • Configurable to your searching needs
  • Search using boolean and search operators AND OR NOT ( )
  • Search using wildcard operatiors E.G. search*
  • Search using special characters E.G. i++;
  • Controllable using a secure API
  • Search thousands of repositories and millions of files quickly
