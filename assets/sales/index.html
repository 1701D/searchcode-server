What searchcode server does:

searchcode server indexes HEAD of any branch of your git or svn repositories.

It then allows you to search over this code quickly and filter by repositories and languages. A partial list of features and the roadmap is included below.

  • Index any git/svn or file repository public or private
  • Identifies 100+ languages and ignores minified code
  • Filter by repository, language or user
  • Estimates the cost of every file with the COCOMO calculation and adjustable to your needs
  • Configurable to your searching needs
  • Search using boolean and search operators AND OR NOT ( )
  • Search using wildcard operatiors E.G. search*
  • Search using special characters E.G. i++;
  • Update repositories using a secure API
